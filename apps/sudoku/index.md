# 할만한 스도쿠

# ⭐ App 소개

- 이 게임은 개발자와 개발자의 가족이 직접 하기 위해서 만들었습니다.
    - 개발자가 출퇴근 길에 지하철에서 약간의 두뇌 활동을 위해 만들었습니다.
    - 개발자의 자녀 또한 같이 틈새 시간에 즐기기 위해서 만들었습니다.
    - 자녀의 무분별한 게임을 제한하고자, 플레이 타임의 제한이 생겼습니다.
- 무료의 다양한 앱도 있지만 플레이 도중 나타나는 과도한 광고에 지쳐서 직접 만들게 됐습니다.
